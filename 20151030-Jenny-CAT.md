给出实数 x,和自然数 m
S=x+x^2/2!+x^3/4!+x^4/6!+......+x^m/(2m-2)!

```python
def factorial(n):
        if n < 0:
                return 1
        result = 1
        for i in range(n):
                result *= i+1
        return result

x = input("Please input x:\n")
m = input("Please input m:\n")

S = 0
for i in range(m):
        S += pow(x,i+1)/float(factorial(2*(i+1)-2))

print S
```
